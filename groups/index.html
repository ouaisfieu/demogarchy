// MEDIAWATCH WAR ROOM

Bienvenue dans l'observatoire citoyen des flux médiatiques. Cette interface permet le monitoring en temps réel des narratifs dominants et l'analyse sémantique de la couverture informationnelle.

// PHILOSOPHIE

Les médias construisent la réalité sociale par le choix des mots, des angles et des silences. Cette war room est un outil de contre-surveillance informationnelle — observer ceux qui nous observent, analyser ceux qui nous analysent.

  • Transparence — Rendre visible l'invisible
  • Décentralisation — Le savoir appartient à tous
  • Critique — Questionner les évidences

// FONCTIONNALITÉS

  • Flux temps réel — Agrégation des publications récentes
  • Analyse sémantique — Cartographie des mots-clés dominants
  • Métriques live — Statistiques de couverture
  • Console système — Logs et commandes avancées
  • Distribution thématique — Répartition par sujet
  • Sources monitorées — État des flux entrants

// COMMANDES TERMINAL

  • help — Affiche l'aide
  • status — État du système
  • clear — Efface la console
  • export — Exporte les données (simulation)
  • scan [mot-clé] — Lance une analyse ciblée
